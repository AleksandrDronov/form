## Используемые технологии и решения
- Typescript
- ReactJS

### Запуск проекта:

1. Клонировать репозиторий.

   git clone git@github.com:AleksandrDronov/form.git

2. Войти в папку проекта.

   cd form

3. Установить зависимости.

   npm install

4. Запуск.

   npm start
